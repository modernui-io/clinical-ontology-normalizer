### Validation Criteria
1. Every recommendation_id has exactly one entry in recommendation event store
2. Every clinician action links to a valid recommendation_id
3. Outcome linkage (when available) uses correct patient_id + time window
4. No orphaned actions (actions without corresponding recommendations)
5. No duplicate recommendation-action pairs
