## Simulation Validation Plan

### Design: Role Fairness and Disparity Risk Check

| Parameter | Value | Rationale |
|-----------|-------|-----------|
| Simulation type | Retrospective replay on historical decisions | No live patient exposure during validation |
| Sample size | 100 simulated decisions per role profile | P4-011-D requirement |
| Comparison method | Non-inferiority against uniform baseline | Must demonstrate "no worse" not "better" |
| Outcome measures | Safety event rate, false positive rate, clinician override rate | Core safety dimensions |
| Non-inferiority margin | -5% (lower CI bound) | Any profile with safety rate >5% worse is rejected |
| Statistical method | Two-proportion z-test with Bonferroni correction for multiple profiles | Control for multiple comparisons |

### Simulation Execution Template

| Step | Action | Expected Result | Pass Criteria |
|------|--------|----------------|---------------|
| 1.1 | Load historical decision dataset (minimum 100 per role) | Dataset loaded with role labels | N >= 100 per profile |
| 1.2 | Apply uniform baseline policy | Baseline gate outcomes computed | All decisions evaluated |
| 1.3 | Apply role profile policy | Profile-specific gate outcomes computed | All decisions evaluated |
| 1.4 | Compute safety outcome delta per profile | Per-profile comparison vs baseline | Deltas calculated |
| 1.5 | Run non-inferiority test | p-value and CI computed | Lower CI bound >= -5% |
| 1.6 | Generate disparity report | Per-profile disparity assessment | No unresolved flags |
