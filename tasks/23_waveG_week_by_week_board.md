# Wave G Week-by-Week Board

Window
- `2026-10-05` to `2026-11-27`

Source
- `tasks/21_waveG_post_wave_closeout_plan.md`

Objective
- Run Wave G as a controlled closeout program with day-level execution across reliability, security, clinical safety, interoperability, support, and 2027 planning.

## Daily Checklist (Use Every Business Day)

- [ ] Review burn-in metrics (latency, error rate, dependency health) before 10:00 local time.
- [ ] Confirm no new Sev1 and triage all open Sev2 with owner and due date.
- [ ] Execute standup by workstream owners (`WG-01`..`WG-06`).
- [ ] Update status of in-scope `WG-*` subtasks in board artifact.
- [ ] Record blocker/decision changes in run log and risk register.
- [ ] Verify evidence links were added for any item moved to `done`.
- [ ] Reconfirm escalation path for any clinical safety or security signal.
- [ ] Publish end-of-day summary (completed, slipped, blocked, next-day focus).
